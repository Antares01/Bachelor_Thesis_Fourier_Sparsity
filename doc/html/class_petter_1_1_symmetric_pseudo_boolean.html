Detailed Description

template<typename real>
class Petter::SymmetricPseudoBoolean< real >

Represents a symmetric pseudo-boolean function of degree <= 4.

This holds a symmetric pseudo-boolean function; it is typically only used when minimizing another pseudo-boolean function. It can only be created from an existing pseudo-boolean function.
