Detailed Description

template<typename real>
class Petter::GeneratorPseudoBoolean< real >

Represents a symmetric submodular pseudo-boolean function of degree <= 4.

This holds a symmetric pseudo-boolean function; it is typically only used when minimizing another pseudo-boolean function. The internal representation is based on the generators specified in the constructor. It can only be created from an existing pseudo-boolean function.
